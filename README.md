CRM
============

Esse projeto foi criado com intuito de estudos e casos de uso.
Seguindo os principios SOLID e Clean Architeture.

Server
----------
Servidor desenvolvido utilizando:
* Express
* Sequelize
* Postgres

Client
---------
Client desenvolvido utilizando:
* React js
* Material UI
* Context API

Devops
----------
* Docker Compose
* Nginx
* Github Actions
